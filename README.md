# 2518班级学术周报网页系统

## 项目简介

这是一个为2518班级设计的学术周报网页系统，提供定期更新的学术内容管理平台。系统采用现代化的前端技术栈，具备响应式设计，支持电脑和手机端的良好显示。

## 主要功能模块

### 1. 读后感/观后感专区
- 学生提交读书笔记和观影心得
- 支持标签分类和内容展示
- 便捷的提交和浏览体验

### 2. 法学领域最新动态速递
- 法学领域最新资讯和动态
- 分类展示（司法解释、立法动态等）
- 实时更新的法学信息

### 3. 精选文献推荐与解析
- 法学专业书籍推荐
- 详细的内容解析和评分
- 支持多种文献类型

### 4. 师生互动交流区
- 学术话题讨论平台
- 实时交流和互动功能
- 师生互动的便捷渠道

### 5. 学术沙龙活动预告与回顾
- 学术活动信息发布
- 在线报名和管理功能
- 活动回顾和总结

## 技术特性

### 前端技术栈
- **Vue 3** - 现代化前端框架
- **Element Plus** - UI组件库
- **Vue Router** - 路由管理
- **Vite** - 快速构建工具

### 响应式设计
- 支持PC端和移动端
- 自适应布局
- 触摸友好的交互体验

### 数据管理
- 本地存储管理
- 数据导入导出功能
- 自动备份机制

## 安装和运行

### 环境要求
- Node.js 16.0+ 
- npm 7.0+

### 安装步骤

1. **克隆项目**
```bash
cd 班级周报
```

2. **安装依赖**
```bash
npm install
```

3. **开发模式运行**
```bash
npm run dev
```

4. **构建生产版本**
```bash
npm run build
```

5. **预览构建结果**
```bash
npm run preview
```
