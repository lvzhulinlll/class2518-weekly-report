# 班级周报系统 - 快速部署指南

## 🚀 最简单的方法：使用GitHub网页上传

### 步骤1：创建GitHub仓库
1. 访问 https://github.com
2. 注册/登录账号
3. 点击右上角"+" → "New repository"
4. 填写信息：
   - **Repository name**: `class2518-weekly-report`
   - **Description**: `2518班级学术周报系统`
   - 选择 `Public`
   - 勾选 `Add a README file`
5. 点击"Create repository"

### 步骤2：上传文件（无需命令行）
1. 在仓库页面，点击"Add file" → "Upload files"
2. 打开文件资源管理器，进入 `C:\Users\lvzhu\Desktop\班级周报`
3. 选择所有文件（Ctrl+A），拖拽到GitHub上传区域
4. 点击"Commit changes"

### 步骤3：开启GitHub Pages
1. 点击仓库页面的"Settings"选项卡
2. 左侧菜单选择"Pages"
3. 设置：
   - **Branch**: `main`
   - **Folder**: `/ (root)`
4. 点击"Save"
5. 等待1-2分钟，显示绿色"Your site is live at..."

### 步骤4：获取访问地址
- **网站地址**: `https://您的用户名.github.io/class2518-weekly-report`
- **教师编辑**: 在上述地址后添加 `/teacher-editor.html`
- **学生查看**: 在上述地址后添加 `/student-view.html?gistId=您的GistID`

## 📱 分享链接格式
部署成功后，分享给学生：
```
https://您的用户名.github.io/class2518-weekly-report/student-view.html?gistId=xxxxxxxxxxxx
```

## 🔧 后续更新
更新内容时：
1. 在本地修改文件
2. 回到GitHub仓库，点击"Add file" → "Upload files"
3. 重新上传修改后的文件
4. 自动更新网站
