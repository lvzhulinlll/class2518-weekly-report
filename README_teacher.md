# 2518班级学术周报 - 教师使用指南

## 📖 项目简介

这是一个专门为教师设计的班级周报管理系统，提供**一键编辑、即时分享**功能，让教师能够轻松管理班级学术内容，学生无需登录即可查看。

## 🚀 快速开始

### 方式一：直接使用教师编辑版
1. 打开 `teacher-editor.html` 文件
2. 开始编辑内容，所有更改会自动保存
3. 复制分享链接给学生

### 方式二：部署到云服务
1. 将文件上传到云服务（如 GitHub Pages、Netlify、Vercel 等）
2. 获得永久访问链接
3. 分享链接给学生

## 📋 使用流程

### 第一步：内容编辑
1. 打开 `teacher-editor.html`
2. 在对应模块中添加内容：
   - **读后感/观后感**：标题、作者、内容
   - **法学动态**：新闻标题、来源、内容
   - **文献推荐**：书名、作者、出版社、简介
   - **学术沙龙**：活动标题、时间、主讲人、描述

### 第二步：内容预览
1. 编辑区域会实时显示添加的内容
2. 可以随时修改或删除
3. 查看统计信息确认内容完整性

### 第三步：分享链接
1. 点击"生成新链接"按钮
2. 复制生成的分享链接
3. 通过微信群、邮件等方式分享给学生

### 第四步：学生访问
1. 学生点击链接即可访问 `student-view.html`
2. 查看最新内容，无需任何操作
3. 支持全文展开阅读

## 🌐 部署指南

### 本地使用
直接双击 `teacher-editor.html` 文件即可在浏览器中打开使用。

### 云服务部署
推荐使用以下云服务：

#### GitHub Pages（免费）
1. 创建 GitHub 仓库
2. 上传所有文件
3. 开启 GitHub Pages 功能
4. 获得 `https://用户名.github.io/仓库名` 链接

#### Netlify（免费）
1. 注册 Netlify 账号
2. 拖拽文件上传
3. 获得永久链接

#### Vercel（免费）
1. 注册 Vercel 账号
2. 连接 GitHub 仓库
3. 自动部署

### 自定义域名（可选）
如果需要更专业的链接，可以绑定自定义域名。

## 🔧 故障排除

### 常见问题

**Q: 学生看不到最新内容？**
A: 确保学生刷新页面，系统会自动同步最新数据。

**Q: 编辑内容丢失？**
A: 系统自动保存，如果清除浏览器缓存可能导致数据丢失。

**Q: 分享链接无效？**
A: 确保文件已正确部署到服务器，链接需要指向实际的网页文件。
